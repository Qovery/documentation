<AccordionGroup>
  <Accordion title="IAM permissions for AWS Parameter Store">
    ```
    {
      "Version": "2012-10-17",
      "Statement": [
        {
          "Effect": "Allow",
          "Action": [
            "ssm:GetParameter",
            "ssm:GetParameters",
            "ssm:GetParametersByPath",
            "ssm:DescribeParameters"
          ],
          "Resource": "*"
        }
      ]
    }
    ```
    <Warning>
      You can restrict access to specific resources using wildcards.

      However, `ssm:DescribeParameters` must always target `Resource: "*"` — AWS does not support resource-level restrictions for this action. Without it, Qovery cannot list available parameters when configuring an external secret.

      Example:
      ```
      {
        "Version": "2012-10-17",
        "Statement": [
          {
            "Effect": "Allow",
            "Action": [
              "ssm:GetParameter",
              "ssm:GetParameters",
              "ssm:GetParametersByPath"
            ],
            "Resource": "arn:aws:ssm:eu-west-3:XXXXXXXXXXXX:parameter/staging*"
          },
          {
            "Effect": "Allow",
            "Action": [
              "ssm:DescribeParameters"
            ],
            "Resource": "*"
          }
        ]
      }
      ```
    </Warning>
  </Accordion>
