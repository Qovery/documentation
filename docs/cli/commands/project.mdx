## Overview

Manage projects within your Qovery organization.

## Commands

### List Projects

List all projects in current organization:

```bash
qovery project list
```

**Flags:**

| Flag | Description |
|------|-------------|
| `--organization` | Organization Name |
| `--json` | JSON output |

### Manage Project Variables

The `project env` subgroup provides commands to manage project-level environment variables and secrets.

```bash
qovery project env list --project "my-project"
```

**`project env list` flags:**

| Flag | Description |
|------|-------------|
| `--organization` | Organization Name |
| `--project` | Project Name (required) |
| `--show-values` | Show env var values |
| `--pretty-print` | Pretty print output |
| `--sort` | Sort environment variables by key |
| `--json` | JSON output |

## Examples

### List Projects

```bash
# Set organization context
qovery context set

# List projects
qovery project list

# List projects as JSON
qovery project list --json
```

### List Project Variables

```bash
qovery project env list --project "my-project"

# Show values
qovery project env list --project "my-project" --show-values
```
